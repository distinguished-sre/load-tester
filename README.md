# Путь в автоматизацию и нагрузочное тестирование с нуля

Одностраничный гайд для человека без опыта в IT, который хочет освоить две ценные инженерные специализации: автоматизацию тестов (AQA/SDET) и нагрузочное тестирование (performance). Объясняет, что это за работа и зачем, показывает путь обучения по шагам и даёт готовый текст для нейросети, с которого начинается практика.

Открыть гайд: https://distinguished-sre.github.io/load-tester/

## Что внутри
Профессия простыми словами, карта из семи этапов (тонкая база и быстро в код: автотесты, метрики, нагрузка, CI/CD) и промпт, который превращает сильную нейросеть в личного преподавателя.

## Публикация
Статическая страница на GitHub Pages, деплой через GitHub Actions при каждом push в `main`.
