# Путь в тестирование с нуля

Одностраничный гайд для человека без опыта в IT. Объясняет, что это за профессия и зачем она, показывает путь обучения по шагам и даёт готовый текст для нейросети, с которого начинается практика.

Открыть гайд: https://distinguished-sre.github.io/load-tester/

## Что внутри
Профессия простыми словами, карта из шести этапов обучения и промпт, который превращает сильную нейросеть в личного преподавателя.

## Публикация
Статическая страница на GitHub Pages, деплой через GitHub Actions при каждом push в `main`.
